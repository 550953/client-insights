# Bachata DanceFit — Аналитика и план развития

**Сеть инструкторов бачаты по России и СНГ**  
735 действующих инструкторов, 40+ городов, 4 уровня обучения.

## 🔍 Что мы поняли о бизнесе

- Это не просто школа, а **экосистема подготовки инструкторов**
- Основной продукт — обучение (₽31-40k) с возможностью оплаты 50/50
- Инструкторы получают лицензию и ведут классы под брендом
- Нет классических роялти — монетизация через ретренинги и продление лицензий

## ⚠️ Текущие проблемы

- NocoDB нестабилен, ломается при автообновлениях
- Данные есть, но не связаны с кабинетом инструктора
- Ручное добавление сертификатов (Яндекс.Диск)
- Нет автоматических напоминаний об оплате

## 💡 Наше решение

✅ Стабильный REST API на реальных данных  
✅ Документация Swagger  
✅ 3 варианта развития (от 50 000 ₽)  
✅ Уже работающее демо: [46.149.75.170:8000/docs](http://46.149.75.170:8000/docs)
