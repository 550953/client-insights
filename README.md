# Client Insights

Аналитика, презентации и технические демо для бизнеса.

## 📁 Структура

| Папка | Описание |
|-------|----------|
| [`bachata-dancefit/`](./bachata-dancefit) | Проект сети инструкторов Bachata DanceFit — анализ бизнеса, данные, API |

## 🧠 Что внутри

- Глубокий разбор бизнес-модели клиента
- Работающая демо-версия API на реальных данных
- Визуальная презентация для встреч
- Дорожная карта развития (3 варианта под любой бюджет)

## 🚀 Как использовать

1. Перейти в папку проекта
2. Открыть `presentation.html` в браузере
3. Изучить бизнес-модель и варианты развития
